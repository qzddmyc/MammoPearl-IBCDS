# 项目优势

1. AI 问答

在本项目的 AI 问答过程中，使用了异步轮询（Polling）架构，通过前端的持续查询对 AI 回复进行获取。

后端中，采用基于本地 JSON 文件作为状态机、结合线程池异步处理的架构，消除了异步的传染性。

2. mutex

在对文件的读写操作、处理页面中关联的长时间任务的按钮中，添加了互斥锁，保证任务的稳定。

3. 加密传输

在对用户信息的传输中，采用 AES-128-CBC 加密算法，双端采用同一套逻辑进行加密与解码，保障了用户的数据安全。

在对用户的昵称保存中，简要地生成了一段自定义的伪 token，避免对用户名的直接展示。

4. 响应式

该项目使用了完善的响应式，保证了在多数设备上页面的展示效果。

5. 数据驱动视图

基于数据驱动实现部分页面的内容展示，简化了页面内容的文档化修改流程，如技术优势页面。

6. 日志

项目采用日志化管理，所有重要操作均产生日志，便于问题的审查与记录的追溯。
